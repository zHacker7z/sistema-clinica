# 🏥 Sistema de Atendimento Inteligente para Clínicas (PROJETO ACADÊMICO)

## 🚀 Demo

🔗 Frontend: https://seu-frontend.vercel.app  
🔗 Backend: https://seu-backend.onrender.com

## 🧠 Sobre o projeto

Sistema fullstack para gestão de consultas médicas com autenticação, controle de horários e integração com APIs externas.

## ⚙️ Funcionalidades

- Cadastro e login com JWT
- Agendamento com verificação de conflitos
- Painel administrativo
- Integração com CEP (ViaCEP)
- Integração com clima (Open-Meteo)
- Fallback resiliente para APIs externas

## 🛠️ Tecnologias

- Node.js + Express
- Vue.js
- MongoDB
- JWT + bcrypt

## ☁️ Deploy

- Backend: Render
- Frontend: Vercel

## 🔐 Segurança

- Autenticação JWT
- Hash de senha com bcrypt
- Proteção de rotas por roles

## 💡 Diferenciais

- Sistema resiliente com fallback para APIs externas
- Cache de requisições
- Estrutura organizada (Controller/Service)
